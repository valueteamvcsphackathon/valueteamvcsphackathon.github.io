Project Overview

Flooding is a critical and recurring challenge in Lagos, particularly in coastal LGAs like Eti-Osa. The Flood & Climate Risk Ledger is a geospatial intelligence platform designed to monitor, analyze, and alert stakeholders about flood risk in near-real time. Combining satellite data, digital elevation models, population data, and parcel boundaries, we produce dynamic risk scores and automated alerts targeted to landowners and local authorities.

Key Features

  • Real-time flood detection using Sentinel-1 SAR imagery.
  • Risk scoring per parcel based on inundation, elevation, and population exposure.
  • Interactive web map dashboard (powered by Mapbox).
  • Automated SMS / email alerts to owners for high-risk parcels.
  • Open source platform hosted on GitHub Pages.
